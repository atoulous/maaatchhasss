# Matcha

This is an match dating app (tinder like) made with React and Express

## Installation

```shell
nvm install 9.2.0
nvm use

yarn || npm i
yarn insert-data || npm run insert-data
yarn start || npm run start

dev:
yarn start-dev || npm run start-dev
```
